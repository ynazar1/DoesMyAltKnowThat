# DoesMyAltKnowThat? — Personal Modification

Personal tweaks and improvements to the original addon.

All credit goes to the original author:

- [Original Addon page](https://www.curseforge.com/wow/addons/doesmyaltknowthat)

---

## Changes

- Made addon cross x-server
- Instead of displaying character name just display "Profession & Status"
  - This prevents insanely large character table
  - Just dump the recipe into warbank, and figure it out later to be honest
- Status is counted all known professions and servers in db:
  - If all known: "Already Known"
  - If partially known: "X of Y Partial"
  - If all unknown: "X of Y Unknown"
